##驗證Git提交
該項目使用[DCO]（https://developercertificate.org/）。 請務必使用-s標誌或
加 `Signed-off-By: Name<Email>` 在提交消息中。 例：
```bash
git commit -s -m 'Informative commit message'
```

##單元測試
提交ART的其他單元測試時，為了保持代碼庫的可維護性，請確保每個單元
測試可以在幾秒鐘內理想地運行。
